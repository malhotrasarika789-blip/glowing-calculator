#  Glowing Calculator

A simple and modern calculator built using **HTML, CSS, and JavaScript**.

This project performs basic arithmetic operations and has a glowing neon UI design.

---

## 🚀 Live Demo

https://malhotrasarika789-blip.github.io/glowing-calculator/

---

##Features

- Neon glowing user interface
- Basic arithmetic operations (+, -, *, /)
- Keyboard support
- Mobile responsive design
- Smooth button animations

---

##Technologies Used

- HTML
- CSS
- JavaScript
